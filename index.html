Download Extension ZIP
Install (Chrome)
  1. Download the ZIP, then unzip it.
  2. Open chrome://extensions.
  3. Enable Developer mode (top right).
  4. Click Load unpacked.
  5. Select the unzipped folder internal-chrome-axtree-extension.
  6. Pin the extension and toggle Enabled in its popup.
